# mp3-folder-player

**按資料夾瀏覽、播放本地 MP3 音樂** — 支援升降 Key、去人聲（伴唱模式）、歌詞編輯，基於 PHP + Web Audio API 開發。

👉 **線上展示**：[https://mhs.ddmmbb.org/mp3/](https://mhs.ddmmbb.org/mp3/)

## ✨ 特色功能

| 功能 | 說明 |
|------|------|
| 📁 **按資料夾瀏覽** | 自動掃描 `mp3/` 下的所有子資料夾，每個子資料夾即為一個音樂分類 |
| 🎛️ **升降 Key** | 即時調整音高，範圍 -12 到 +12 半音，唱歌或聆聽更自由 |
| 🎤 **去人聲（伴唱模式）** | 一鍵消除人聲，保留伴奏，適合卡拉 OK 或練習 |
| 📝 **歌詞編輯與同步** | 內建歌詞編輯器，播放時歌詞自動向上滾動同步 |
| 🔁 **多種播放模式** | 單曲播放、單曲循環、全部循環、隨機播放 |
| 📤 **上傳 MP3** | 透過網頁將 MP3 上傳至選定的資料夾 |
| 🎨 **視覺化介面** | 毛玻璃風格、動態光暈、響應式設計 |

## 🛠️ 技術棧

- **後端**：PHP（檔案掃描、上傳、歌詞讀寫）
- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **音訊處理**：Web Audio API + AudioWorklet（實時升降 Key、去人聲）
- **播放器**：HTML5 `<audio>` 元素

## 📂 目錄結構

```
your-web-root/
├── index.php                 # 主程式
├── player.js                # 前端播放器邏輯
├── style.css                # 樣式表
├── soundtouch-processor.js  # 升降 Key 處理器（AudioWorklet）
└── mp3/                     # ⭐ 音樂目錄（需手動建立）
    ├── 華語流行/            # 子資料夾 = 播放清單分類
    │   ├── 歌曲01.mp3
    │   ├── 歌曲02.mp3
    │   └── 歌曲01.txt       # 歌詞（與 mp3 同檔名，自動載入）
    ├── 搖滾經典/
    └── 絕地戰兵/            # 展示網站中的範例資料夾
        ├── 外送民主01.mp3
        └── 外送民主02.mp3
```

## 🚀 安裝步驟

### 1. 下載專案
```bash
git clone https://github.com/ddmmbb-2/mp3-folder-player.git
cd mp3-folder-player
```

### 2. 建立音樂目錄
在專案根目錄下建立 `mp3` 資料夾：
```bash
mkdir mp3
```

### 3. 放入音樂檔案
在 `mp3/` 下建立子資料夾（例如 `華語流行`、`絕地戰兵`），將 `.mp3` 檔案放入子資料夾中。

*（可選）歌詞檔案：與 MP3 同目錄、同主檔名，副檔名為 `.txt`，例如 `歌曲01.txt`。*

### 4. 設定網站
將整個專案上傳至網站伺服器（需支援 PHP），確保 `mp3/` 資料夾及子資料夾具備讀取權限。

### 5. 開啟瀏覽器
訪問 `http://你的網址/index.php` 即可使用。

## 📖 使用說明

### 播放音樂
- 從右上角下拉選單選擇音樂資料夾
- 點擊右側播放清單中的歌曲即可播放
- 使用底部控制按鈕：播放/暫停、上一首、下一首

### 播放模式
點擊模式按鈕（循環圖示）可切換：
- 🔂 **單曲循環**：重複播放同一首歌
- 🔁 **全部循環**：播完整個清單後從頭開始
- 🎲 **隨機播放**：隨機選取下一首
- ▶️ **單曲播放**：播完後停止

### 升降 Key
拖曳「音調 (Key)」滑桿，範圍 -12 到 +12 半音，即時調整音高。

### 去人聲（伴唱模式）
勾選「開啟去人聲 (伴唱模式)」，人聲頻段會被抑制，保留伴奏。

### 編輯歌詞
1. 播放歌曲後，點擊右下角「✏️ 編輯歌詞」按鈕
2. 在彈出視窗中輸入或貼上歌詞
3. 按「儲存變更」即可
4. 下次播放同一首歌時，歌詞會自動載入並同步滾動

### 上傳 MP3
- 頁面底部有上傳表單，選擇 MP3 檔案後按「上傳至此資料夾」
- 上傳的目標資料夾為目前選中的資料夾（由右上角下拉選單決定）

## ⚙️ 設定調整

若需修改音樂目錄位置，請編輯 `index.php` 開頭的 `$mp3_base_dir` 變數：

```php
$mp3_base_dir = __DIR__ . "/mp3";  // 可改為其他絕對路徑
```

## 🧪 開發者備註

- **升降 Key 處理器**：`soundtouch-processor.js` 需與 `play.php` 放在同一目錄，該檔案實作 AudioWorklet 進行實時音高調整。
- **瀏覽器相容性**：建議使用最新版 Chrome、Edge、Firefox（需支援 AudioWorklet）。
- **自動播放政策**：部分瀏覽器限制未經使用者互動的自動播放，播放前需點擊頁面任一處。
